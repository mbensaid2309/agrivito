# Agrivito - Brand Name Decision

## Objectif

Ce document officialise le choix du nom de marque du produit.

Le projet était initialement nommé **AgriAI** comme nom de travail.

Après réflexion sur le positionnement, la crédibilité, la mémorisation et l'ambition internationale, le nom produit retenu est :

```text
Agrivito
```

---

# Décision finale

## Nom produit retenu

```text
Agrivito
```

## Ancien nom de travail

```text
AgriAI
```

## Nom du repository officiel

```text
agrivito
```

## Repository GitHub officiel

```text
https://github.com/mbensaid2309/agrivito.git
```

---

# Pourquoi changer de AgriAI

Le nom **AgriAI** était clair, mais trop générique.

## Points forts de AgriAI

* simple ;
* compréhensible ;
* évoque directement agriculture + intelligence artificielle.

## Limites de AgriAI

* trop générique ;
* difficile à différencier ;
* difficile à protéger comme marque ;
* moins mémorable ;
* trop centré sur la technologie IA ;
* moins adapté à une marque produit sérieuse et durable.

---

# Pourquoi Agrivito

## Points forts

**Agrivito** est retenu car le nom est :

* professionnel ;
* simple à retenir ;
* agricole ;
* sérieux ;
* internationalisable ;
* compatible avec une application mobile ;
* compatible avec une entreprise SaaS ;
* moins générique que AgriAI ;
* plus distinctif que les noms purement descriptifs.

---

# Positionnement du nom

Agrivito doit porter l'image suivante :

* une plateforme agricole intelligente ;
* un assistant fiable pour les décisions agricoles ;
* un outil professionnel pour les agriculteurs ;
* une marque sérieuse, moderne et mémorisable ;
* une solution utilisable au Maroc, en Afrique, puis à l'international.

---

# Promesse associée

La promesse produit reste :

```text
Le meilleur ingénieur agronome que chaque agriculteur peut avoir à ses côtés.
```

Agrivito n'est pas simplement une app IA.

Agrivito est une plateforme intelligente d'assistance à la décision agricole.

---

# Alternative de secours

Le nom alternatif retenu en cas de problème majeur de disponibilité est :

```text
Cultivio
```

Cultivio reste une bonne alternative car il est :

* professionnel ;
* agricole ;
* mémorisable ;
* adapté à une application mobile ;
* sérieux.

Mais le choix prioritaire reste :

```text
Agrivito
```

---

# Réserve importante

La décision produit est validée, mais la disponibilité légale doit encore être confirmée.

Avant lancement public, il faudra vérifier :

* disponibilité du nom de domaine ;
* disponibilité des réseaux sociaux ;
* disponibilité App Store ;
* disponibilité Play Store ;
* recherche Google approfondie ;
* recherche marque au Maroc ;
* recherche marque en Europe si expansion prévue ;
* risque de confusion avec des noms proches.

---

# Règles de nommage à partir de maintenant

À partir de cette décision :

## Dans les nouveaux documents

Utiliser :

```text
Agrivito
```

Ne plus utiliser AgriAI comme nom produit principal.

## Dans le code

Utiliser :

```text
agrivito
```

Exemples :

```text
agrivito-backend
agrivito-mobile
agrivito-api
```

## Dans le repository

Le repository officiel est :

```text
agrivito
```

## Dans les prompts Codex

Les prompts Codex doivent désormais parler de :

```text
Agrivito
```

et non plus de :

```text
AgriAI
```

---

# Impact sur les documents existants

Les documents `01` à `22` peuvent encore contenir le nom **AgriAI**, car ils ont été rédigés avant la décision de branding.

Cependant, pour éviter la confusion :

* les futurs documents utiliseront **Agrivito** ;
* les prompts opérationnels utiliseront **Agrivito** ;
* le README principal utilisera **Agrivito** ;
* le code utilisera **agrivito** ;
* une passe de renommage pourra être faite plus tard si nécessaire.

---

# Impact sur Codex

Codex doit comprendre que :

```text
AgriAI = ancien nom de travail
Agrivito = nom produit officiel
```

Toutes les nouvelles implémentations doivent utiliser **Agrivito**.

Le prompt Sprint 1 doit être mis à jour pour refléter ce changement.

---

# Décision CTO

Le nom **Agrivito** est validé comme nom produit officiel.

Le nom **AgriAI** est conservé uniquement comme historique de travail.

Le projet passe maintenant en mode exécution sous le repository :

```text
https://github.com/mbensaid2309/agrivito.git
```
